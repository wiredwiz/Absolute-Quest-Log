# AbsoluteQuestLog

Absolute Quest Log is a World of Warcraft library add-on for use when you want to retrieve quest log data or react to certain important quest events.  It abstracts away all the ugly bits and rough 
edges of the wow questlog and quest information APIs and standardizes them to a single set of APIs that don't change.  Currently this library only targets Burning Crusade Anniversary edition, but
I hope to add support for other versions later.
